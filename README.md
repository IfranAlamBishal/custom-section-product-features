# 🎯 Custom Product Features Section
## Overview

The Custom Product Features Section is a flexible, visually engaging layout designed to highlight key product benefits or brand features. It displays feature blocks on both sides of a central image, creating a balanced and premium presentation.

This section is ideal for showcasing product advantages, ingredients, technology, or brand values in a structured and modern format.

Perfect for:

  - Health & longevity products (ingredients, benefits)
  
  - Skincare and beauty brands
  
  - Tech or gadget features
  
  - Premium product storytelling
  
  - Landing pages and product highlights
  

The section is fully customizable via the Shopify Theme Editor and optimized for all devices.

# ✅ How to Install

Follow these steps to install and use the Custom File Upload section in your Shopify theme:

## 1. Open Your Shopify Theme Code Editor

- From your Shopify admin, go to: Online Store → Themes

- Find your active theme, then click ⋯ (More) → Edit code.

## 2. Create a New Section

  - Navigate to the Sections folder

  - Click Add a new section

  - Name it (e.g. custom-product-features.liquid)

  - Paste the full code

  - Click Save

# ✅ How to Use

## 1. Add Section to Your Page

  - Go to Online Store → Themes → Customize

  - Navigate to your desired page (Home / Product / Landing page)

  - Click Add section

  - Select Custom Product Features

## 2. Add Feature Blocks

  - Each feature is added as a block.

  - For each block, you can configure:

  - Column position (Left / Right)

  - Icon (optional)

  - Title

  - Subheading (optional)

  - Description text

  - Icon position (left, right, or top)

  - Text alignment (desktop & mobile)

  - Custom spacing (padding)

# 🎨 Section Settings
 
  - Header Settings

  - Section Heading

  - Subheading

  - Font size, weight, and color

  - Alignment (left / center / right)

  - Spacing below heading

  - Center Image Settings

  - Upload the central image

  - Control image width

  - Enable/disable floating animation

  - Typography & Content Styling

  - Icon size

  - Feature title font size & weight

  - Description text size & color

  - Subheading styles (per block)

  - Layout & Spacing

  - Background color

  - Section padding (top & bottom)

  - Grid layout (auto responsive)

# 🛒 How It Works (Layout Flow)

   Features are split into left and right columns

   A center image acts as the visual anchor

   Each feature block includes:

  - Icon
  - Title
  - Optional subheading
  - Description

 ## On mobile:

  Layout automatically stacks into a single column
  
  Alignment adjusts based on mobile settings

# 🧩 Tips

Keep feature text short and clear for better readability

Use consistent icon style for a premium look

Limit to 4–6 features for best visual balance

Use the center image to highlight your main product

Enable floating animation for a modern feel

# ⚠️ Troubleshooting

  - Layout looks unbalanced?
  → Ensure equal number of blocks on left and right columns
  
  - Icons not showing?
  → Check if the icon image is uploaded in the block
  
  - Alignment issues on mobile?
  → Adjust mobile text alignment in block settings
  
  - Image too large or small?
  → Adjust “Center Image Width” in section settings
